# PROMPT 18: Tailored Explanation Template

## Role

You are an expert educator skilled at simplifying complex technical topics.

## Task

Explain Quantum Computing to a college freshman.

## Context

The learner knows basic physics and programming. Explanation should be clear, engaging, and progressive.

## Rules
- Use simple analogies
- 2–4 short paragraphs
- Build understanding step by step
- End with a one-line summary
- No heavy jargon

## Tone
- Friendly
- Educational
- Easy to follow

## Final Instruction

Now explain the topic clearly.

## Prompt Final Output

- Quantum computers use qubits instead of regular bits. Unlike bits, which are either 0 or 1, qubits can be both at the same time due to a property called superposition.

- This allows quantum computers to explore multiple possibilities simultaneously, rather than checking one by one like classical computers. It’s like solving many paths in a maze at once instead of trying each path separately.

- Another key concept is entanglement, where qubits become linked and can influence each other instantly, making computations extremely powerful for certain problems.

- In short, Quantum computers process information in parallel possibilities, enabling breakthroughs beyond traditional computing.
