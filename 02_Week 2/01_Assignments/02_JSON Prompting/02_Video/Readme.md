
# 🎬 AI Video Generation using JSON Prompting
## 📌 Overview

This project demonstrates how structured JSON prompting can be used to create cinematic, story-driven AI-generated videos.

The focus is on generating a horror dinosaur creature sequence with controlled scenes, camera movement, and visual effects.

## 🎯 Objective

## To compare:

- Basic video generation using normal prompts
- Advanced video generation using structured JSON prompts

## 🔹 Normal Prompt Example

A horror dinosaur demon walking in a dark swamp

## 🔹 JSON Prompt Example
{
  "subject": "mutated dinosaur demon with glowing magma cracks and multiple eyes",
  "environment": "dark foggy swamp with dead trees",
  "scenes": [
    "empty swamp with fog",
    "glowing eyes appear",
    "creature emerges slowly",
    "close-up of face",
    "creature roars and spreads wings"
  ],
  "camera": "cinematic tracking and close-ups",
  "lighting": "dark with red glow",
  "effects": ["fog", "ash", "glow"],
  "style": "cinematic horror"
}
